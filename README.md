# Physical-Reservoir-Computing
Final year capstone project for the University of Pretoria. Topic: Pysical Reservoir computing, implemented through acoustic reverberation.

Library packages used for this code:
  numpy
  scipy
  matplotlib.pyplot

Claude Sonnet 4.6 was used in the creation of this code as an assitant, largely for the purposes of making plots and code legible. All code has been checked and cross referenced.
